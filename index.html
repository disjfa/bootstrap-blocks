Bootstrap blocks

So what do we have here. These are a set of basic blocks that can be used in websites using bootstrap. On every page there is an example and a code block. The text in the codes are identical, as they use the same html. So hopefully they work.

The things we use are:

  • Bootstrap for the styling.
  • Font awesome for the icons.
  • unsplash for the images.
  • eleventy for building this page.
  • github for hosting.

If you have any ideas, tips, tricks and such please let us know on github. If you would like to add a template please create a codepen and let us know. And alway be excelent to eachother.
